Product Features

Why This Series Is Used

Retracting Claw Locking

The claw-locking structure helps reduce thread damage and improves connection efficiency during repeated testing.

316 Stainless Steel Body

The stainless steel body provides strong mechanical performance and high pressure capability.

Multiple Actuation Routes

Manual and pneumatic operation concepts can support different bench, fixture, and production requirements.

Large Internal Flow Path

A larger internal flow path helps reduce influence on running tests and functional validation.

Serviceable Face Seal

The face seal design makes seal replacement easier for routine maintenance.

Operation Logic

Three-Step Connection Flow

1. Press the Handle

For the G85L handle version, press the handle so the claws extend and remain expanded.

2. Push to the Threaded Port

Push the connector toward the threaded interface until the thread end face fully contacts the main seal.

3. Release to Clamp and Seal

Release the handle so the metal claws retract, clamp the external thread, and complete the sealed connection.
